Welcome to UX : Xoops Cube Legacy

The main goal of UXCL is to develop an end-user patch/package with a common User Interface and User Experience enhanced for the Content Management System XOOPS Cube Legacy - Web Application Platform.

UI Design

UXCL has a folder common/ui with some handsome CSS/JS Frameworks used for Themes and Templates. Go ahead and continue to Themes to browse through them. You can easily download the samples to customize the design of your website.
