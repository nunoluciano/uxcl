Outline

  • This module provides Private Messages functions for XOOPS Cube 2.2.
  • Default user profile can be replaced by the template of this module.
  • Add Block to display messages of Inbox and Sentbox.
  • Messages status, read, unread, lock.
  • Search messages by status or username from Inbox and Sentbox.
  • Option to purge messages automatically.
  • Settings to define pagination (messages per page).
  • Forward message to email account.
  • Blacklist management.

Preferences

  • Administrator can replace template user_userinfo.html, define default number of messages per page, delay to purge messages and set if unread messages are also deleted, use of block 'New Messages'.
