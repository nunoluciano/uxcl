User module - Blocks in the public

Login
Overview

This block shows the login-form for only anonymous users who still are not passing the authentication. If the user registration is allowed, it shows the link to register.php. And, if the SSL login is allowed, it shows the link specified by options of preferences.

Note

You can do login in user.php without this block, if you has uninstalled this block by mistake.

Who's Online
Overview

This block shows a name of users who may be accessing to your site and numbers of them.

Note

Getting correct informations for online users is too heavy process. For that, this block doesn't try to find the correct results. But, it's enough to make your site exciting.

New Members
Overview

This block shows a name of users who are registered to your site recently.

Options
Display x members
Specifies limit number of displaying recently users.
Display member avatars
In case of YES, displays user's avatars on the name of the users.
Top posters
Overview

This block shows the ranking of the top posters.

Options
Display x members
Specifies limit number of displaying users.
Display member avatars
In case of YES, displays user's avatars on the name of the users.
Do not display users whose rank is x
You can remove users belonging to specific groups from the ranking.
